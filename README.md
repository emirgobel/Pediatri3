# Pediatri3

Bu repo, paylaştığınız ekran tasarımlarını temel alan tek sayfalık pediatrik arayüz uygulamasını içerir.

## Çalıştırma

Statik bir arayüzdür. Aşağıdaki komutla yerelde açabilirsiniz:

```bash
python3 -m http.server 4173
```

Ardından tarayıcıdan `http://localhost:4173` adresine gidin.
